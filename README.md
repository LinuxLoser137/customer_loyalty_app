# Customer Loyalty App

A beginner-friendly customer loyalty web app designed to help users view rewards, coupons, profile information, and dashboard features.

## Live Website

melodious-creponne-417f4b.netlify.app

## About This Project

This project was created to practice front-end web development and app-style page organization. The app simulates a customer loyalty system where users can view rewards, coupons, profile details, and admin-style features.

The goal of this project was to practice building a more complete website with multiple sections and user-focused features.

## Features

- Demo login concept
- Customer dashboard
- Rewards section
- Coupons section
- Profile section
- Admin-style features
- Clean navigation
- App-style layout

## Technologies Used

- HTML
- CSS
- JavaScript
- Git
- GitHub

## What I Learned

While building this project, I practiced structuring a larger website, organizing multiple features, using JavaScript for basic interactivity, and thinking about how users move through an application. I also practiced creating a project that feels more like a real software product instead of only a static webpage.
